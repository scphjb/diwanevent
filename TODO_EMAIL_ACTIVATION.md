# TODO: تفعيل نظام البريد الإلكتروني الاحترافي 📧

هذه القائمة للمتابعة غداً لإتمام ربط الدومين وتفعيل الإرسال التلقائي للرسائل.

## 1. إعداد حساب SendGrid
- [ ] إنشاء حساب على [SendGrid](https://sendgrid.com/).
- [ ] إنشاء **API Key** جديد (بصلاحيات Full Access).
- [ ] البدء في عملية **Sender Authentication** لإثبات ملكية الدومين الخاص بك.

## 2. إعدادات Cloudflare (DNS)
- [ ] إضافة سجلات الـ **CNAME** الثلاثة التي سيعطيك إياها SendGrid.
- [ ] **تنبيه هام:** تأكد من جعل حالة السحابة (Proxy) **رمادية (DNS Only)** لهذه السجلات تحديداً.
- [ ] إضافة سجل **SPF** (من نوع TXT) إذا طلب ذلك لضمان عدم وصول الرسائل للـ Junk.

## 3. ربط المنصة (الخلفية)
- [ ] فتح ملف `attendance_system/.env`.
- [ ] وضع البريد الرسمي في خانة `EMAILS_FROM_EMAIL`.
- [ ] وضع الـ API Key في خانة `SMTP_PASSWORD`.
- [ ] حفظ الملف وإعادة تشغيل السيرفر.

## 4. الاختبار والتحقق
- [ ] القيام بعملية تسجيل وهمية لمشارك جديد ببريد إلكتروني حقيقي.
- [ ] التحقق من وصول رسالة "تأكيد التسجيل" والباركود.
- [ ] التحقق من وصول رسائل "إعادة تعيين كلمة المرور" للمسؤولين.
